## 构建

1. 安装[Git](https://git-scm.com/)

2. 安装[rustup](https://rustup.rs/)

3. 构建

```
git clone https://github.com/University-Of-Fool/uof-status-client-rust.git && cd uof-status-client-rust/
cargo build --release
```
